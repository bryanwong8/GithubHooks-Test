## 5-1-1 Step 4
### name
```
Solution 
```
### md_content
```
Here is the full solution:
```
### code_snippet

```python
@property
def is_solved(self):
    solved = True
    for row in self:
    	for cell in row:  
            if (cell.is_visible or cell.is_mine):
            	continue
       		else:
            	solved = False      
	return solved
```

Here is an alterative, more compact solution using the `all` keyword:

### code_snippet

```python
@property
def is_solved(self):
    return all((cell.is_visible or cell.is_mine) for row in self for cell in row)	
```
