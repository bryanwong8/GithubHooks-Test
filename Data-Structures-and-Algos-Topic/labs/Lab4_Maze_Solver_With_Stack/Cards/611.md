Main function contains the following steps:

Define files:

```python
	mazeFile = "maze.txt"
	doneFile = "done.txt"
```

Load files:

```python
	maze= loadFile(mazeFile)
	done= loadFile(doneFile)
```

Print Maze and solve maze:

```python
	printGrid(maze)
	solveMaze(maze)
```

Print Done:

```python
	print("\n")
	printGrid(done)
```
